AI-Powered Learning Disability Detection App

Skill Sprout is a gamified mobile application designed to help detect early signs of learning disabilities in children during their academic stages. The app provides engaging Duolingo-style assessment games across key domains such as Dyslexia, Dyscalculia, Dysgraphia, ADHD, and Memory Challenges.

Children complete fun interactive tasks, while parents and teachers receive detailed performance analytics, category-wise scores, and AI-based risk predictions powered by a Random Forest model.
